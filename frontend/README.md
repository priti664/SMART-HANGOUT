# SMART HANGOUT FINDER

An attractive AI-powered multi-page frontend for finding the best hangout places in Mumbai with budget filters, categories, offers, ratings, reviews, addresses, and an AI chatbot.

## Features

- **Multi-Page Website**: Home, Search, Places, and AI Chat pages
- **User Authentication**: Login and registration modals
- **Smart Search**: Filter by budget range (min/max) and category (All, Cafe, Restaurant, McDonald's, Mall, Other)
- **Detailed Place Info**: Ratings, reviews, Mumbai addresses, available items, and offers
- **AI Chatbot**: Interactive assistant with location-based recommendations
- **Attractive UI**: Modern design with gradients, animations, and responsive layout
- **Mumbai Focus**: Real addresses in Andheri, Kandivali, Malad, Goregaon, Borivali
- **Rich Content**: 35+ places with real images, items like burgers, pizza, fries, etc.

## Pages

- **index.html**: Home page with features overview and category links
- **search.html**: Search form with budget and category filters
- **places.html**: Browse all places with detailed information
- **chatbot.html**: Dedicated AI assistant page

## How to Run

1. Open any HTML file in a modern web browser (start with `index.html`).
2. No server required - it's a static multi-page frontend.

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter, gradients)
- JavaScript (ES6+)
